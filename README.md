# Wokwi LCD Character Editor

A simple editor for creating custom LCD characters for Wokwi simulations.

<img src="assets/demo.png" alt="Demonstration" width="800">

## Installation

### Quick installation from GitHub

```bash
pip install git+https://github.com/jokelbaf/wokwi-lcd-char-editor.git
```

Then run:

```bash
wokwi-char
```

### Manual installation

1. Clone the repo:
```bash
git clone https://github.com/jokelbaf/wokwi-lcd-char-editor.git
```
2. Install deps:
```bash
uv sync
```
3. Run:
```bash
uv run main.py
```
